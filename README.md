WatermarkKit is a browser-based tool for batch watermarking images. The app lets you drop in multiple images and apply a watermark to all of them at once. You can change the opacity, scale, and placement using a 9-point grid. 
It also includes an invisible checkerboard pattern option and supports exporting to JPEG, PNG, WebP, or AVIF.

The entire project is contained in the watermarkKit.html file. It uses vanilla JavaScript, CSS, and HTML with no external libraries or dependencies needed.
To use it, open the HTML file in any browser, drag in your photos and a PNG logo, adjust the sidebar settings, and click Process All Images to download the results.
